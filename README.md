Descrição: Fract'ol é um projeto de computação gráfica que envolve a renderização de belos fractais (Mandelbrot e Julia) em uma janela gráfica. O projeto exige a utilização da biblioteca MiniLibX para gerenciamento de janelas e renderização de pixels.

Desafios e Implementação:

Matemática Fractal: Compreender as fórmulas complexas que geram os conjuntos de Mandelbrot e Julia e mapeá-las para pixels na tela.

Renderização: Desenvolver a lógica para pintar pixels individualmente e atualizar a tela eficientemente.

Gerenciamento de Eventos: Implementar hooks para eventos de teclado (movimentação, mudança de fractal, cores) e mouse (zoom).

Bônus Implementados:

Zoom com Posição do Mouse: O zoom segue a posição real do mouse, proporcionando uma experiência de usuário intuitiva.

Movimentação da View: Permitir o deslocamento da visualização usando as teclas de seta.

O que aprendi:

Fundamentos de Computação Gráfica: Como pixels são renderizados e manipulados para criar imagens.

MiniLibX: Utilização de uma biblioteca gráfica para gerenciamento de janelas, imagens e eventos.

Interação com Usuário: Implementação de controles interativos via teclado e mouse.

Otimização Gráfica: Técnicas para renderizar imagens complexas de forma performática.
